Required Image Assets

1. favicon.png

Specifications:
  • Size: 32x32 pixels (also create 16x16 version)
  • Format: PNG with transparency
  • Colors: Use primary brand color (#2E5BBA)
  • Design: Simple, recognizable at small sizes
  • Suggested: "BA" initials or a simplified gear icon

2. apple-touch-icon.png

Specifications:
  • Size: 180x180 pixels
  • Format: PNG
  • Colors: Use brand colors (#2E5BBA and accent color)
  • Design: Similar to favicon but more detailed
  • Include padding (16px on all sides)
  • Use rounded corners (optional, iOS will round automatically)

3. og-image.jpg

Specifications:
  • Size: 1200x630 pixels (optimal for social sharing)
  • Format: JPG
  • Text: Include site name and tagline
  • Design elements to include:
    • Business Automation Insights
    • 40+ Years Experience
    • Visual element (gear/automation icon)
    • Your name or business logo
  • Use brand colors and maintain good contrast
  • Keep text large and readable at small sizes

Design Notes:

  • Maintain consistent branding across all assets
  • Use the established color palette:
    • Primary: #2E5BBA
    • Accent: #FFA500
    • Background: #FFFFFF
  • Ensure all text is readable
  • Test favicon at multiple sizes
  • Test og-image in social media preview tools

Next Steps:

  1. Create these images using design software (Photoshop, Figma, etc.)
  2. Save in the specified formats
  3. Place in the root directory of the website
  4. Test across different platforms and devices
  5. Update HTML references if needed
